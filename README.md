# KidTime 童心护航

跨设备时间账本 · 儿童时间管理助手

## 核心理念

KidTime 不锁机、不监控。它是一块"时间仪表盘"——通过主动打卡计时、任务悬赏和透支惩罚机制，帮助儿童建立自我管理意识，让家长从"吼叫式提醒"变为"规则式引导"。

## 功能一览

| 模块 | 功能 |
|------|------|
| 时间银行 | 每日自动发放额度，打卡计时每分钟扣减 |
| 环形进度盘 | 绿/黄/红三色预警，剩余时间一目了然 |
| 悬赏任务 | 家长发布任务，儿童主动接取，审核通过发放奖励 |
| 透支惩罚 | 超时或违规扣减，透支债务次日自动抵扣 |
| 超时告警 | 时间耗尽时全屏告警 + 系统闹铃 |
| 家长管理 | PIN 保护，调整额度 / 增扣时间 / 管理任务 / 查看流水 |
| 前台服务 | 息屏切换 App 计时不中断，通知栏持续显示状态 |

## 屏幕概览

**儿童仪表盘**
- 环形进度盘居中，大号数字显示剩余分钟数
- 绿色"开始使用"按钮启动计时，红色"结束使用"停止
- "📋 悬赏任务"入口查看和接取任务
- 右上角齿轮图标进入家长管理（需 PIN）

**家长管理**
- 当前状态卡片（额度 / 余额 / 欠款）
- 每日额度 ±5 步进调整
- 额外增加时间 / 执行惩罚扣减
- 悬赏任务管理（创建 / 批准 / 驳回）
- 修改 PIN 码 / 查看时间流水

**超时告警**
- 全屏红色背景 + ⏰ 图标
- 系统默认闹铃循环播放
- "我已关闭电视/平板"按钮停止

## 技术架构

```
Java 8  ·  MVVM  ·  Room Database  ·  SharedPreferences
Material Design 3  ·  Foreground Service  ·  Canvas Custom View
```

| 层 | 组件 |
|----|------|
| 数据 | `PrefsManager` + Room (`TimeTransaction`, `TaskRecord`) |
| 业务 | `TimeBankManager` — 每日结算、奖惩、任务审批 |
| 服务 | `TimerForegroundService` — 60s 定时扣减、超时告警 |
| 视图 | `CircleTimerView` — 环形进度盘 Canvas 绘制 |
| UI | `ChildDashboardFragment` + `ParentSettingsActivity` + 辅助页面 |
