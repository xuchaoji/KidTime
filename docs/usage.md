# KidTime (童心护航) 使用指南

## 概述

KidTime 是一款为家长设计的儿童设备时间管理工具，通过"时间银行"机制帮助儿童管理电子设备使用时间：

- 每日自动发放时间额度
- 完成任务可获得奖励时间
- 家长可手动增加 / 惩罚扣减时间
- 超时使用产生透支，次日自动抵扣

## 安装与启动

1. 将 APK 安装到儿童使用的设备上
2. 首次启动进入儿童仪表盘
3. **默认家长 PIN：`1234`**，默认每日额度：**30 分钟**
4. 建议安装后立即进入家长管理修改 PIN 和额度

## 儿童端操作

### 仪表盘

- **环形进度盘** — 直观显示剩余时间比例：

  | 颜色 | 比例 | 含义 |
  |------|------|------|
  | 绿色 | ≥ 50% | 时间充裕 |
  | 黄色 | 20% ~ 50% | 需注意 |
  | 红色 | ≤ 20% | 即将耗尽 |

- 进度盘中心显示剩余分钟数，余额超出额度时环形盘自动扩容

### 开始 / 结束计时

- 点击绿色 **"开始使用"** → 启动前台计时服务，每 60 秒自动扣 1 分钟
- 点击红色 **"结束使用"** → 停止服务，余额冻结
- 计时中可息屏、切换 App，计时不中断
- 按钮下方显示 "🔴 计时中…" 状态提示

### 接取悬赏任务

- 点击仪表盘 **"📋 悬赏任务"** 查看家长发布的任务
- 点击任务右侧绿色 **"接取"** 按钮申请完成
- 接取后任务变为"待审核"，由家长批准发放奖励

### 时间耗尽告警

- 余额归零 → 计时自动停止 → 弹出全屏红色告警页 + 系统闹铃循环
- 点击 **"我已关闭电视/平板"** 停止铃声并退出

## 家长管理

### 进入方式

仪表盘右上角齿轮图标 → 输入家长 PIN → 进入管理页面

### 状态概览

顶部卡片实时显示：每日额度、当前余额、透支欠款

### 每日额度调整

- 点击 **+5 / −5** 按钮以 5 分钟步进
- 调高额度时，**当前余额同步增加**等量分钟数（如 30→35，余额也 +5）
- 调低额度时，余额同步减少（不低于 0）
- 变更记录自动写入时间流水

### 额外增加时间

- 点击 **"额外增加时间"**
- 输入分钟数和原因（如"今天表现好奖励 10 分钟"）
- 增加的时间仅当天有效，次日跨天结算时不保留

### 修改 PIN 码

- 点击 **"修改家长PIN码"**
- 输入当前 PIN → 新 PIN → 确认

### 执行惩罚扣减

- 点击 **"执行惩罚扣减"**
- 输入扣减分钟数和原因
- 余额不足时自动产生透支欠款，次日从额度扣除

### 悬赏任务管理

- 点击 **"悬赏任务管理"** 进入

**创建任务**：
1. 点击右上角 ＋ → 输入名称和奖励分钟数 → 添加
2. 任务发布后显示为 **"待接取"**

**审核任务**：
- 儿童接取后任务变为 **"待审核"**
- 点击 **"批准"** → 自动发放奖励 + 写入流水
- 点击 **"驳回"** → 退回"待接取"状态

### 查看时间流水

点击 **"时间流水记录"** 查看全部变动明细：

| 类型 | 说明 |
|------|------|
| 每日发放 | 每日结算自动发放 |
| 每日额度调整 | 家长调整额度时的余额变动 |
| 任务奖励 | 批准任务发放 |
| 额外奖励 | 家长手动增加 |
| 使用消耗 | 计时每分钟扣除 |
| 违规惩罚 | 家长扣减 |

---

## 每日结算机制

- 首次启动计时 / 进入家长管理时触发结算
- 公式：**实际发放 = 每日额度 − 透支欠款**
- 透支超过额度时当日余额为 0，剩余欠款滚动至次日

---

## 通知栏

计时期间通知栏持续显示 **"KidTime正在计时中"**，停止计时后自动消失。

---

## 当前局限

- 无数据导出 / 云端同步
- 仅支持单设备单儿童场景
